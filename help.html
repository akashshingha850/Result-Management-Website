1. To Add a data:

  • Create a Database first
  • Add a table to the database
  • Then click to add data. After giving the numbers the gpa will be added automaticly. Make sure that the roll and name is valid or the data will not be added.

    • 2. To update a data

  • Click on the update data
  • Enter the roll of which data you want to update.
  • Then change the data you want to update

    • 3. To Delete a data

  • First search by the roll of which data you want to delete. Make sure the roll is listed on the database.
  • then click on delete. Be sure because this can not be undone.
